GIT CHEAT CODE

1. Check Status (Who is newer?)

Compare your laptop to the GitHub server.

STEP 1: FETCH LATEST INFO
git fetch origin
STEP 2: CHECK STATUS
git status
  • "Ahead of origin/main": Laptop is NEWER. (Need to Push)
  • "Behind origin/main": GitHub is NEWER. (Need to Pull)
  • "Up to date": Perfect sync.

2. Make Online = Local (Push)

Use this after you finish coding on your laptop.

git add .
git commit -m "Update"
git push origin main

3. Make Local = Online (Pull)

Use this if you edited code on the GitHub website directly.

git pull origin main
